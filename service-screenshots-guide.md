# 📸 서비스 화면 가이드

## 🌐 사이트 주소
https://library-bid-clean.vercel.app/

## 📋 캡처해야 할 화면

### 1. 메인 화면 (초기 상태)
- URL: https://library-bid-clean.vercel.app/
- 포함 요소:
  - 제목: "📚 국립중앙도서관 · 국회도서관 공고 정보"
  - "🔍 최신 공고 조회" 버튼
  - 빈 상태 안내 메시지
  - 카테고리 카드 (전체/국립중앙도서관/국회도서관/키워드)

### 2. 로딩 중 화면
- 버튼 클릭 후 로딩 상태
- 포함 요소:
  - "🔄 조회 중..." 버튼 (비활성화)
  - 노란색 로딩 박스
  - "데이터 로딩 중..." 스피너

### 3. 데이터 표시 화면
- 공고 목록이 로드된 상태
- 포함 요소:
  - 카테고리별 건수 (전체/국립중앙도서관/국회도서관/키워드)
  - 검색창
  - 정렬 옵션
  - 공고 카드 목록 (색상별 구분)

### 4. 필터링 화면
- 특정 카테고리 선택 시
- 포함 요소:
  - 선택된 카테고리 (예: 국립중앙도서관)
  - 필터링된 공고 목록

### 5. 검색 화면
- 키워드 검색 사용 중
- 포함 요소:
  - 검색창에 입력된 키워드
  - 검색 결과 공고 목록

## 📝 신청서 작성 팁

### 서비스 설명
```
국립중앙도서관과 국회도서관의 입찰공고를 실시간으로 조회하고,
키워드(도서관, 기록물, DB, 디지털 등) 기반 필터링 기능을 제공하는
입찰정보 통합 모니터링 서비스
```

### 주요 기능
1. 실시간 입찰공고 조회
2. 기관별 필터링 (국립중앙도서관/국회도서관)
3. 키워드 기반 검색
4. 최신순/오래된순 정렬
5. 공고 상세 페이지 링크 제공

### 사용 API
- 조달청_나라장터 입찰공고정보서비스
- Operation: getBidPblancListInfoServcPPSSrch
- 조회 항목: 용역 입찰공고 (최근 1개월)
